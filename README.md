# NeoPiXXL
XXL version of the classic WS2812B/NeoPixel with same functionality.


## Assembly

### BOM:
 - Driver:
    - PCB
    - WS2811
    - 33 ohm 0603 resistor x2
    - 100 ohm 0603 resistor
    - 100 nF 0603 capacitor
 - Plate:
    - PCB
    - Red LED
    - Green LED
    - Blue LED
 - Hardware:
    - Bottom plate (3D printed)
    - Top plate (vacuum formed)
